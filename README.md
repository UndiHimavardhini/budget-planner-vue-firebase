# Budget Planner Web Application

A web-based Budget Planner application designed to help users track income, expenses, and savings goals with interactive data visualizations.

This project was developed as part of a learning and internship-oriented web development exercise. Guidance and step-by-step assistance were taken during development to better understand Vue.js, Firebase integration, and real-world project structure.
